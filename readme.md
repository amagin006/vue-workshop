## ０. 始める前に - 環境の構築
実際にVueを使い始める前にプロジェクトを作成し開発環境を用意しましょう。
PCには既にNode.jsが勿論インストールされていますよね？
念の為チェックしておきましょう。
```
$ node -v
v10.15.0
```
この様に表示されればNodeは既にインストールされています。
